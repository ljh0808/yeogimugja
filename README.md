# Spring Boot 프로젝트
**YeogiMugja** : 지역에 따른 숙박업체 플랫폼 웹 서비스

## 프로젝트 정보

주제 선정 사유 : 급증하는 여행수요에 따른 숙박 웹서비스 제공
- 여행수요가 갈수록 증가하는 추세, 숙박 웹서비스는 특정업체만 존재하는 상황

**개발 목표** 

- 기본적인 숙박 검색 및 예약 서비스 제공
- 숙소,객실등록 서비스 제공
- 사용해보고 싶었던 기술(API,Security 등)을 최대한 적용

## 개발 기간

- **시작일**: 2024.05.13
- **종료일**: 2024.06.05 (총 4주)

## 팀원 및 주요 기능

1. **김헌종** - 메인페이지 Front-end 제작 담당 / [RbCream](https://https://github.com/RbCream) 
2. **유재명** - 웹 디자인, 숙소예약시스템 담당 / [Jamie-Nestor90](https://github.com/Jamie-Nestor90)
3. **이준호** - 로그인 및 회원가입, 관리자 숙소 관리, 결제서비스 담당  / [ljh0808](https://github.com/ljh0808)
4. **한재엽** - 숙소 검색결과 및 상세페이지 담당 / [sunniesfish](https://github.com/sunniesfish)
